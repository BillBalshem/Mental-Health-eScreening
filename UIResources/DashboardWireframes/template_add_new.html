For demo purpose please follow the below steps

 

Add Text Block

Step 1. Select Block Type (TEXT)

Step 2. Copy text from VistA (Example Text 1: Click here)

Step 3: Insert Variables from Variable list

Step 4: Click Insert button to add the block to the template (Select Step_1_text)

 

Add If condition

Step 5: Build this condition "If Condition | (dep_score_phq2) (greater than) (2) OR | (dep_score_phq9) (greater than) (9)

" by Select Type if

Step 6: Select Variable dep_score_phq2 then greater than then enter value 2

Step 7: Select OR then Select Variable dep_score_phq9 then greater than then enter value 9

Step 8: Click Insert button to add the block to the template (Select Step_2_if)

 

Add Text Block

Step 1. Select Block Type (TEXT)

Step 2. Copy text from VistA (Example Text 2: Click here)

Step 3: Insert Variables from Variable list

Step 4: Click Insert button to add the block to the template (Select Step_3_text)

 

Add Else If condition from main template page - Add button

Step 1: Select Else

Step 2: Click Insert button to add the block to the template (Select Step_4_text)

 

Add Text Block

Step 1. Select Block Type (TEXT)

Step 2. Copy text from VistA (Example Text: No.)

Step 3: Click Insert button to add the block to the template (Select Step_5_text)
